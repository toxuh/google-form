# Google form

Refactoring of [this](https://github.com/F3ll3n/Form-with-Google-Sheets) project.

#### Start:

1. Install dependencies

```bash
# with npm:
npm i

# with yarn
yarn
```

2. Create .env file with your Google script id

```bash
VITE_G_KEY=YOUR_ID_HERE
```

3. Start application

```bash
# with npm:
npm run dev

# with yarn
yarn dev
```
